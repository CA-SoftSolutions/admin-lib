ğŸ“– API REFERENCE
================================================================================

SEOTracker.init(config)
  Initialize the library
  Required: apiUrl, projectId
  Optional: debug, trackClicks, trackScroll, trackVisibility, trackUnload

SEOTracker.track(eventType, details)
  Track custom event
  Example: SEOTracker.track('purchase', { amount: 99.99 })

SEOTracker.flush()
  Manually send queued events
  Returns: Promise

SEOTracker.setOptOut(value)
  Set user opt-out preference
  Example: SEOTracker.setOptOut(true)

SEOTracker.destroy()
  Clean up and destroy instance

SEOTracker.version
  Get library version
  Returns: "1.0.0"
